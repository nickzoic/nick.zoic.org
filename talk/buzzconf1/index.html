Programming the ESP8266

github: themadinventor/esptool

  • NodeMCU makes this totally easy
  • USB cable and esptool
  • Otherwise, serial converter

esp-open-sdk

github: pfalcon/esp-open-sdk

  • C development using GCC toolchain
  • Quite a lot of libraries for WiFi

libesphttpd

github: Spritetm/esphttpd

  • Simple HTTP / Websocket handling
  • Flash-based read-only filesystem

NodeLua

nodelua.org

  • Load with esptool
  • Interpreted languages on a uC!
  • REPL over serial port
  • Cut-down standard libraries
  • Libraries representing hardware
