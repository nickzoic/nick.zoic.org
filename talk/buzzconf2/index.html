MicroPython I/O

>>> import machine
>>> import time
>>> pin2 = machine.Pin(2)
>>> pin2.init(machine.Pin.OUT)
>>> while True:
...     pin2.low()
...     time.sleep(0.5)
...     pin2.high()
...     time.sleep(0.5)

What doesn't MicroPython have?

  • An operating system
  • Multiprocessing
  • Much memory to play with
  • Debugger / Profiler

Summary

If you're interested in Microcontrollers / IoT / Robotics:

  • ESP8266 is about the cheapest
  • MicroPython is about the easiest
